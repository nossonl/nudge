# ReinforceClaw OpenClaw Plugin

OpenClaw plugin bridge for ReinforceClaw feedback.

Install this package through OpenClaw or npm when you want OpenClaw channels such as Telegram, WhatsApp, Slack, or Discord to send `/rl good`, `/rl bad`, `/rl undo`, and `/rl status` feedback to your local ReinforceClaw bridge.

The Python CLI and trainer are published separately on PyPI as `reinforceclaw`.
